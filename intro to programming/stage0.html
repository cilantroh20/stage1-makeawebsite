Important Concepts

About the Web:

The web is basically just a bunch of computers communicating with eachother. It starts with you, and your computer. You tell your computer "Hey take me here." This is done by entering the URL into your browser. Your browser then sends a HTTP Request (Hypertext Transfer Protocol) to the server. The server then responds with the HTML documents that the user requested, and the web browser interprets the page and displays it to the user.

HTML:

HTML stands for Hypertext Markup Language. HTML is the basic foundation of the web, the web is basically a collection of HTML documents. HTML documents are made up of text content: "what you see", and markup:"how it looks" . For further information, check this out.

HTML Tags and Elements:

HTML documents contain HTML elements. HTML elements are composed of an opening tag("This is the beginning of a heading"), content("what you want in between the tags"), and a closing tag("This is the end of a heading"). Think of an HTML Tag as telling the browser i want this content to look like this, put this there, make it do this. Here are some references of an HTML Element and an HTML Tag

Inline VS Block:

HTML elements are either inline or block .

Inline elements usually just contain text.

Whereas, Block elements make an invisible box around text.

These types of elements are important because they help you avoid using white spaces(white spaces interpret tabs/multiple spaces as just one space), so it is great to use line breaks and block elements to start a fresh line!

Document Structure:

HTML documents are structured with the following tags so the browser can indicate the proper structure of the document.

<!DOCTYPE html >
< html >
< head >
< title > content < /title >
< /head >
< body >
content
< /body >
< /html >
